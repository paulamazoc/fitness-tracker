## Getting Started

Clone this repo and run the following scripts to install dependencies and start development server:

```bash
npm install
npm start dev
```

Navigate to [http//localhost:5173](http//localhost:5173). App will be available by default on this port.

And to run the test:

```bash
npm run test
```
